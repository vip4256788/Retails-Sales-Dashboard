# Retail Sales Dashboard – Power BI

## Overview
This project analyzes retail sales data and provides insights into sales trends, product categories, customer demographics, and transaction patterns. The dashboard is designed to help business managers make data-driven decisions.
## Dataset Columns
- Transaction ID
- Age
- Age Group
- Gender
- Product Category
- Price Per Unit
- Quantity
- Total Amount
## Key KPIs
- **Total Sales** – Total revenue from all transactions  
- **Total Transactions** – Number of individual transactions  
- **Total Quantity Sold** – Sum of all products sold  
- **Average Sale per Transaction** – Average revenue per transaction
## Dashboard Layout
- **Top Section:** KPI cards showing Total Sales, Total Transactions, Total Quantity, and Average Sale  
- **Middle Section:** Sales by Product Category (Bar Chart) and Sales by Gender (Donut Chart)  
- **Bottom Section:** Sales by Age Group (Bar Chart) and Quantity by Category (Bar Chart)  
- **Interactive Filters:** Gender, Age Group, Product Category
## Tools Used
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- GitHub (for version control and project sharing)
